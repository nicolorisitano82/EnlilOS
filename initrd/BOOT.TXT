Mount profile bootstrap:
/        -> initrd cpio (ro)
/dev     -> devfs
/data    -> ext4 rw-full su virtio-blk quando disponibile
/sysroot -> ext4 rw-full su virtio-blk quando disponibile

Entry points:
/INIT.ELF      -> launcher login shell del bootstrap
/ARKSHBOOT.ELF -> selftest ELF non interattivo per la login shell
/bin/arksh     -> shell di login v1 con fallback automatico a /bin/nsh
/bin/nsh       -> recovery shell userspace
/usr/bin/loadkeys  -> cambio layout tastiera (`us`/`it`)
/usr/bin/kbdlayout -> mostra il layout tastiera attivo
/etc/arkshrc   -> rc globale bootstrap
/etc/vconsole.conf -> layout tastiera bootstrap (`KEYMAP=...`)
/home/user/.config/arksh/arkshrc -> rc utente fallback
/usr/share/kbd/keymaps/us.map -> keymap bootstrap US
/usr/share/kbd/keymaps/it.map -> keymap bootstrap IT
